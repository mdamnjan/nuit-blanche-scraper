# nuit-blanche-scraper
A Python/Selenium scraper that grabs exhibit information from the Toronto 2023 Nuit Blanche website and preps it to be imported into `Google Maps`.

## How to use it with Google Maps

### Import into Google Maps
To use it with Google Maps, import the generated `exhibits.csv` file into [Google My Maps](https://www.google.com/maps/d/u/0/) and when/if prompted select the `Address` column as the column that provides location data.  

### Accessing it in Google Maps

- Go to the `Saved` tab in the app and select `Maps` (on the bottom right)
- Select `Nuit Blanche TO 2023`

Click on an exhibit to see details about it or get directions

### Removing it from your map
To remove the map from `Google Maps` entirely, you'll need to log into [Google My Maps](https://www.google.com/maps/d/u/0/).


If you just want to stop seeing the map in your app, click on any of the exhibit icons and select `View Map Legend`, and then `Close`. You can always open it again through the `Saved` tab, clicking on `Maps`.
